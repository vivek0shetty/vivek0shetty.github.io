My Work

Tourist Guide Booking System

The Tourist Guide Booking System is a user-friendly platform that connects travelers with professional guides. It allows tourists to browse, book, and communicate with guides based on location, language, and expertise. The system features real-time availability, secure payments, and user reviews to enhance the booking experience. Designed for seamless navigation, it ensures hassle-free travel planning while promoting local tourism.

Weather Prediction Using LSTM

This project leverages Long Short-Term Memory (LSTM) networks, a type of recurrent neural network (RNN), to predict weather conditions based on historical data. By analyzing temperature, humidity, wind speed, and atmospheric pressure, the model learns temporal patterns to generate accurate forecasts. The system enhances predictive accuracy by handling sequential dependencies, making it valuable for climate research and disaster preparedness. The implementation involves data preprocessing, model training, and performance evaluation to ensure reliable weather predictions.

AI-Driven Multilingual News Aggregator

The AI-Driven Multilingual News Aggregator is an innovative mobile application that collects, summarizes, and translates news articles from various sources in real time. Built using Flutter (Dart) in Android Studio, the app provides users with a seamless reading experience, allowing them to access news in multiple languages. Key features include AI-powered news summarization, language translation, user authentication, and a personalized news feed. This project enhances global news accessibility, breaking language barriers and delivering concise, relevant information to users worldwide.
